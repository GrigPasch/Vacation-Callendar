# Vacation Callendar

An app where company employees can login and submit their desired leave/vacation days.

----------------------------------------------------------------------------------------

# IF YOU CHOSE TO CLONE THE REPO OR DOWNLOAD IT AND RUN IT

## To initialize it for frontend:

npm install (to install necessary dependencies

npm start (to run it at local host)
  
-----------------------------------------------------------------------------------------

## To initialize it for FullStack

cd vacation-backend -> npm install -> npm run dev
  
This will initiate the database connection through the API

cd Vacation-Callendar -> npm install -> npm start
  
This will initiate the app on the corresponding port (3001)

 -----------------------------------------------------------------------------------------

The FullStack is based on a mySQL Local Host and Database I created using MySQL Workbench. 

I am new to this, so I don't know if the full stack will work for anyone else besides me, 
but if I find out how to make it work while deployed or for anyone who downloads/clones the
repo, I will update ASAP.
